# Cisco Packet Tracer és Hálózati Definíciók

| Kifejezés                                      | Meghatározás                                                                                                                                              |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Vezeték nélküli forgalomirányító (router)**                  | Egy eszköz, amely összekapcsolja a router és a vezeték nélküli hozzáférési pont (WAP) funkcióit.                                                           |
| **Vezeték nélküli hozzáférési pont (WAP)**      | Egy hálózati eszköz, amely létrehoz egy vezeték nélküli helyi hálózatot (WLAN).                                                                              |
| **Ethernet**                                   | Egy vezetékes hálózati technológia család, amelyet gyakran használnak helyi hálózatokban.                                                                   |
| **Wi-Fi**                                      | Egy vezeték nélküli hálózati technológia az IEEE 802.11 szabványok alapján.                                                                                  |
| **Sávszélesség**                               | A hálózat vagy internetkapcsolat maximális adatátviteli sebessége.                                                                                          |
| **Késleltetés**                                | Az adatcsomag útjának ideje egyik ponttól a másikig történő utazásához.                                                                                     |
| **QoS (Minőségi Szolgáltatás)**                | Olyan technológiák összessége, amelyek kezelik a hálózati forgalmat annak érdekében, hogy bizonyos adattípusok prioritást élvezzenek.                       |
| **IoT (Internet of Things)**                     | Egy olyan fizikai eszközök (dolgok) hálózata, amelyek érzékelőkkel, szoftverrel és kapcsolódással vannak ellátva, lehetővé téve az adatok gyűjtését és cseréjét az interneten. |
| **PoE (Power over Ethernet)**     | Egy olyan technológia, amely lehetővé teszi, hogy az Ethernet kábelek az adatok mellett elektromos áramot is szállítsanak.                                 |
| **VPN (Virtual Private Network)**               | Egy biztonságos kapcsolat, amely lehetővé teszi a távoli hozzáférést egy magánhálózathoz egy nyilvános hálózaton keresztül (pl. az interneten).                |
| **Port átirányítás**                           | Egy olyan technika, amely lehetővé teszi a külső eszközök számára, hogy hozzáférjenek egy privát hálózaton belül egy adott szolgáltatáshoz.                    |
| **DMZ (Demilitarizált Zóna)**                  | Egy hálózati szegmens, amely elválaszt egy privát hálózatot egy nyilvános hálózattól (pl. az internet).                                                      |
| **Hálózati biztonság**                         | A hálózat védelmének gyakorlata az engedély nélküli hozzáférés, visszaélés, módosítás vagy megsemmisítés ellen.                                             |
| **Felhőszámítás**                              | Az interneten keresztül történő számítási szolgáltatások (pl. szerverek, tárolók, adatbázisok, szoftverek) nyújtása.                                        |
| **Packet Tracer**                              | A Cisco által kifejlesztett hálózati szimulációs szoftver a hálózati koncepciók és protokollok tanulásához.                                                 |
| **Hálózat**                                    | Egy összekapcsolt eszközök csoportja, amelyek kommunikálnak és megosztják az erőforrásokat.                                                                 |
| **Eszköz**                                     | Bármilyen hardver a hálózaton, például számítógép, kapcsoló, útválasztó vagy szerver.                                                                       |
| **Topológia**                                  | A hálózat fizikai vagy logikai elrendezése.                                                                                                                |
| **Protokoll**                                  | Egy szabályok halmaza, amelyek meghatározzák, hogyan kommunikálnak az eszközök.                                                                             |
| **IP-cím**                                     | Egyedi azonosító egy eszköz számára a hálózaton.                                                                                                           |
| **Alhálózati maszk**                           | Meghatározza, hogy az IP-cím mely része azonosítja a hálózatot, és melyik része azonosítja a gépet (eszközt).                                                |
| **Alapértelmezett átjáró**                      | Az útválasztó, amelyet egy eszköz használ a más hálózatokon található eszközökkel való kommunikációhoz (pl. az internet).                                 |
| **Kapcsoló**                                   | Egy hálózati eszköz, amely összekapcsolja az eszközöket ugyanabban a hálózatban, és MAC-címek alapján továbbítja az adatokat.                               |
| **Útválasztó**                                 | Egy hálózati eszköz, amely különböző hálózatokat kapcsol össze, és IP-címek alapján továbbítja az adatokat.                                                 |
| **SOHO (Small Office/Home Office)**                                       | Kis iroda/otthoni iroda, általában kisebb méretű hálózat kevesebb eszközzel és egyszerűbb igényekkel.                                                       |
| **MOTD (Message of the Day)**                       | Egy üzenet, amelyet a felhasználók látnak, amikor bejelentkeznek egy hálózati eszközbe vagy szolgáltatásba.                                                 |
| **MAC-cím**                                    | Egyedi hardverazonosító egy hálózati interfész kártyához (NIC).                                                                                            |
| **VLAN**                                       | Egy virtuális LAN, amely egy fizikai hálózatot több logikai hálózatra oszt fel.                                                                               |
| **NAT (Hálózati címfordítás)**                 | Egy folyamat, amely lehetővé teszi, hogy több eszköz privát IP-címekkel ossza meg egyetlen nyilvános IP-címet.                                            |
| **DHCP (Dinamikus Házigazda Konfigurációs Protokoll)** | Egy protokoll, amely automatikusan hozzárendel IP-címeket és más hálózati konfigurációs beállításokat az eszközöknek.                                   |
| **DNS (Domain Name System)**                   | Egy protokoll, amely fordítja a tartományneveket (pl. [www.pelda.com](https://www.pelda.com)) IP-címekké.                                                 |
| **Tűzfal**                                     | Egy hálózati biztonsági rendszer, amely a biztonsági szabályok alapján figyeli és irányítja a bejövő és kimenő hálózati forgalmat.                         |
| **TFTP (Triviális fájlátviteli protokoll)**     | Egy egyszerű protokoll fájlok átvitelére eszközök között, gyakran használják hálózati eszközök indításához vagy firmware frissítéséhez.                     |
| **SFTP (SSH fájlátviteli protokoll)**          | Egy biztonságos fájlátviteli protokoll, amely az SSH-t használja az adatok titkosításához és hitelesítéséhez.                                             |
| **TCP (Átviteli vezérlő protokoll)**            | Egy kapcsolatalapú protokoll, amely megbízható adatküldést biztosít a kapcsolat létrehozása előtt.                                                       |
| **UDP (Felhasználói adatgram protokoll)**       | Egy kapcsolat nélküli protokoll, amely a sebességet helyezi előtérbe a megbízhatóság helyett, gyakran használják médiafájlok streamingjéhez és online játékhoz. |
| **HTTP (Hipertext átviteli protokoll)**         | A világhálón történő adatátvitelre használt protokoll.                                                                                                      |
| **HTTPS (Hipertext átviteli protokoll biztonságos verziója)** | Az HTTP biztonságos verziója, amely titkosítja az adatokat a lehallgatás és a manipuláció ellen.                                                         |
| **FTP (Fájlátviteli protokoll)**               | Egy protokoll, amelyet a kliens és a szerver közötti fájlok átvitelére használnak.                                                                           |
| **SSH (Biztonságos héj)**                      | Egy biztonságos protokoll a hálózati eszközök távoli eléréséhez és kezeléséhez.                                                                              |
| **WPA2 (Wi-Fi Protected Access 2)**            | Egy biztonsági protokoll a Wi-Fi hálózatokhoz, amely erősebb titkosítást biztosít az idősebb protokollokhoz képest, mint például a WEP.                      |
| **WPA2-PSK (Előre megosztott kulcs)**           | A WPA2 egy módja, amely előre megosztott jelszót használ az azonosításhoz.                                                                                   |
| **WPA2-Enterprise**                            | A WPA2 egy módja, amely RADIUS kiszolgálót használ az azonosításhoz, általában nagyobb hálózatokban használják.                                             |
