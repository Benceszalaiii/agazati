# HTML validátor

**Hogyan használjuk a HTML validátorokat**

1. **Válassz egy validátort:** A legszélesebb körben használt és legmegbízhatóbb validátor a hivatalos W3C Markup Validation Service ([https://validator.w3.org/](https://validator.w3.org/)). Ezen kívül más online validátorok is elérhetők.

2. **Add meg a HTML-ed:** A HTML-ed megadására három fő mód van:
   * **Direkt bevitel:** Illeszd be a HTML kódodat közvetlenül a validátor szövegmezőjébe.
   * **Fájl feltöltése:** Töltsd fel a HTML fájlodat (.html vagy .htm) a validátorba.
   * **URI alapján:** Add meg a weboldalad URL-jét (ha az nyilvánosan elérhető online).

3. **Validálás:** Kattints a "Check" vagy "Validate" gombra.

<Callout type="info">
Megjegyzés: A feladat kérni fogja, hogy készíts egy képernyőképet a validálási eredményről, ezért mindenképpen mentsd el vagy rögzítsd a kimenetet.
</Callout>

### HTML validálása

Itt egy példa egy egyszerű weboldal HTML kódjára:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Random HTML Page</title>
</head>
<body>
    <h1>Welcome to My Random HTML Page!</h1>
    <p>This is a short example of an HTML page generated by a program.</p>
    <p>Feel free to modify and experiment with the code!</p>
</body>
</html>
<h2>Random HTML Page</h2>
<p>This is a random HTML page generated by a program.</p>
<p>Feel free to modify and experiment with the code!</p>
```
