## Dobozmodell

A CSS dobozmodell leírja az elemek elrendezését egy weboldalon. A következő összetevőkből áll:

- **Tartalom:** A doboz tényleges tartalma, például szöveg vagy képek.
- **Belső margó (padding):** A térköz a tartalom és a szegély között.
- **Szegély (border):** A tartalmat és a belső margót körülvevő szegély.
- **Külső margó (margin):** A térköz a szegély és a szomszédos elemek között.

Íme egy példa a dobozmodell működésére:

```css
.box {
  width: 150px;
  height: 150px;

  margin: 30px; /* Külső margó */
  border: 5px solid lightgrey; /* Körülveszi a belső margót és a tartalmat */
  padding: 10px; /* Belső margó */
}
```

<div>

- Fekete: Tartalom
- Szürke: Belső margó
- Világosszürke: Szegély
- Fehér: Külső margó

## CSS betöltési sorrend

Amikor több CSS fájl van linkelve egy HTML dokumentumhoz, a betöltésük sorrendje befolyásolhatja az oldal stílusát. Így működik a betöltési sorrend:
Az utoljára importált CSS fájl felülírja az előzőleg importált fájlokból származó, ütköző stílusokat.

```html
<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="style1.css" />
    <link rel="stylesheet" href="style2.css" />
    # style2.css felülírja style1.css közös stílusait
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
```
