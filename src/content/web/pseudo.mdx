# CSS Pszeudo-szelektorok

| Pszeudo-szelektor | Leírás                                                                               |
| ----------------- | ------------------------------------------------------------------------------------ |
| `:empty`          | Gyermektelen elemekre alkalmaz stílusokat (beleértve a szöveges csomópontokat is).   |
| `:not(selector)`  | Stílusokat alkalmaz azokra az elemekre, amelyek nem egyeznek a megadott választóval. |
| `:disabled`       | Letiltott űrlap elemekre alkalmaz stílusokat.                                        |
| `:enabled`        | Engedélyezett űrlap elemekre alkalmaz stílusokat.                                    |
| `:checked`        | Kiválasztott jelölőnégyzetekre vagy választógombokra alkalmaz stílusokat.            |
| `:valid`          | Érvényes bemenettel rendelkező űrlap elemekre alkalmaz stílusokat.                   |
| `:invalid`        | Érvénytelen bemenettel rendelkező űrlap elemekre alkalmaz stílusokat.                |
| `:in-range`       | Megadott tartományon belüli értékű beviteli mezőkre alkalmaz stílusokat.             |
| `:out-of-range`   | A megadott tartományon kívüli értékű beviteli mezőkre alkalmaz stílusokat.           |
| `:read-only`      | Csak olvasható űrlap elemekre alkalmaz stílusokat.                                   |
| `:read-write`     | Nem csak olvasható űrlap elemekre alkalmaz stílusokat.                               |
| `:optional`       | Opcionális űrlap elemekre alkalmaz stílusokat.                                       |
| `:required`       | Kötelező űrlap elemekre alkalmaz stílusokat.                                         |

## Hivatkozás Pszeudo-szelektorok

| Pszeudo-szelektor | Leírás                                                                               |
| ----------------- | ------------------------------------------------------------------------------------ |
| `:link`           | Még nem látogatott hivatkozásokra alkalmaz stílusokat.                               |
| `:visited`        | Már meglátogatott hivatkozásokra alkalmaz stílusokat.                                |
| `:hover`          | Stílusokat alkalmaz, amikor az egér a hivatkozás fölé kerül.                         |
| `:active`         | Stílusokat alkalmaz, amikor a hivatkozást éppen kattintják (aktív).                  |
| `:focus`          | Stílusokat alkalmaz, amikor a hivatkozás fókuszba kerül (pl. tabulátorral).          |
| `:target`         | Stílusokat alkalmaz arra a hivatkozásra, amely az aktuális URL-fragmentum célpontja. |
