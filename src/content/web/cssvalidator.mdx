# CSS validátor

**Mik azok a CSS validátorok**

A CSS validátorok online eszközök, amelyekkel ellenőrizhető a CSS kódod, hogy az megfelel-e a World Wide Web Consortium (W3C) által meghatározott hivatalos CSS szabványoknak.

**Hogyan használjuk a CSS validátorokat**

1. **Válassz egy validátort:** A legnépszerűbb és legmegbízhatóbb validátor a hivatalos W3C CSS Validator ([https://jigsaw.w3.org/css-validator/](https://jigsaw.w3.org/css-validator/)). Ezen kívül számos más lehetőség is elérhető online.

2. **Add meg a CSS-ed:** A CSS megadására többféle lehetőség is van:
   * **Direkt bevitel:** Illeszd be a CSS kódodat közvetlenül a validátor szövegmezőjébe.
   * **Fájl feltöltése:** Töltsd fel a CSS fájlodat (.css) a validátorba.
   * **URI alapján:** Add meg a CSS fájlod URL-jét (ha az nyilvánosan elérhető a weben).

3. **Validálás:** Kattints a "Check" vagy "Validate" gombra.

<Callout type="info">
Megjegyzés: a feladat kérni fogja, hogy készíts egy képernyőképet a validálási eredményről, ezért mindenképpen mentsd el vagy rögzítsd a kimenetet.
</Callout>

### Példa

Itt egy egyszerű weboldal CSS kódja:

```css
body {
    background-color: #f1f1f1;
    font-family: Arial, sans-serif;
    color: #333;
}

h1 {
    font-size: 24px;
    font-weight: bold;
    color: #ff0000;
}

p {
    font-size: 16px;
    line-height: 1.5;
}

a {
    color: #007bff;
    text-decoration: none;
}

.container {
    max-width: 960px;
    margin: 0 auto;
    padding: 20px;
}
```
